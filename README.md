# GUI vs **CLI**

## CLI의 장점
1. 효율성
2. 정밀제어
3. 자동화
4. 표준환경

### Gitbash를 이용한 CLI의 예시
- `touch` a = a라는 파일 생성
- `mkdir foo = foo`라는 파일 생성
- `cd` = change directory
- `cd ..` = 상위폴더로 이동
- `cd .` = 현재경로를 현재폴더로 이동
- `rm` = remove
- `rm - r` (폴더명) = 폴더 삭제
- `ls` = 내부 폴더 파일 출력
- `pwd` = 현재 경로 출력
