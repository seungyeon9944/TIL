### 형변환 Type Conversion
- 암시적 형변환 (자동으로 데이터 타입을 변환)
ex. 정수 + 실수 = 실수. 불리언 + 정수 = T는 1 + 정수로 계산, 불리언간의 덧셈 T + F = 1 + 0 = 1
- 명시적 형변환 (직접 함수로 지정하여 변환)
